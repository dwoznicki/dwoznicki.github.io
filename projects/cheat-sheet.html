A CSS Postioning Cheat Sheet

CSS positioning is the black sheep of the web development community. It's the thing that everyone loves to hate. And for good reason. There's something almost cryptic about how elements with different styles interact on a page. My goal here is not to completely clear up the mystery, but just to lay down a few guidelines and give some tips for positioning in CSS.

General Positioning Properties

  • text-align:
  • float: and clear:
  • position:
  • display:
  • border:, margin: and padding:

text-align:

As it's name suggests, text-align allows you to define how you want your text to be aligned inside an element.

  • center; centers your text
  • left; aligns text to left
  • right; aligns text to right

float:

The float property defines an element's tendency in relation to its surroundings. It's often used to make text wrap around an image.

  • none; is the default value
  • left; floats the element left
  • right; floats the element right

clear:

Elements after a floating object will naturally flow around it. To stop this, you'd give the clear property to the element(s) after the floater.

  • left; avoids floaters to the left
  • right; avoids floaters to the right
  • both; avoids floaters on both sides

position:

The position property tells an element how you want it to sit on the page. By default, elements follow the flow of the HTML page, which renders them in the order in which they appear in the document. Position lets you change how an element interacts with this flow.

  • static; is the default value
  • relative; moves the element relative to its static position
  • absolute; moves the element relative to its first positioned ancestor
  • fixed; glues the element to a place on the page

display:

When the browser lays out your page, it essentially assigns each element to a block of space. You can change how a block functions using the display property.

  • inline; is the default value
  • inline-block; allows you to give an object a height and width
  • block; forces a line break after the element

border:, margin:, and padding:

These properties affect the amount of space an element takes up. You can use them to position an element on the page.

  • border: gives the element a border, though it's not really used for positioning
  • margin: is the space between the element and its surroundings
  • padding: is the space between the element and its border
